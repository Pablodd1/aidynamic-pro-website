## 🚀 Deployment Instructions

### Step 1: Upload Website (2 minutes)
**Option A: Netlify (FREE)**
1. Go to https://app.netlify.com/drop
2. Drag & drop the `aidynamic-pro/` folder
3. Site goes live instantly
4. Add custom domain: `aidynamic.pro`

**Option B: Vercel (FREE)**
1. Install Vercel CLI: `npm i -g vercel`
2. `cd aidynamic-pro`
3. `vercel --prod`

**Option C: GitHub Pages (FREE)**
1. Push `aidynamic-pro` folder to repo
2. Enable GitHub Pages in settings
3. Select `main` branch, `/root`

### Step 2: Set Up Contact Form (5 minutes)
1. Go to https://formspree.io/register
2. Create account with `info@aidynamic.pro`
3. Get your form endpoint URL
4. Replace `info@aidynamic.pro` in the form action on line ~595 of index.html
5. Test by submitting a message

### Step 3: Set Up Calendly (5 minutes)
1. Go to https://calendly.com
2. Create account
3. Set up 30-minute "Discovery Call" event
4. Get your Calendly link
5. Replace `https://calendly.com/aidynamicpro/discovery` in index.html

### Step 4: Google Analytics (Optional)
1. Go to https://analytics.google.com
2. Create property for aidynamic.pro
3. Get Tracking ID (G-XXXXXXXXXX)
4. Replace in `analytics.js`

### Step 5: Outreach Launch (30 minutes)
1. Sign up for Instantly.ai ($37/month)
2. Warm up 2-3 email domains (2 weeks)
3. Import `outreach-contacts.csv`
4. Customize templates
5. Launch 50 emails/day
